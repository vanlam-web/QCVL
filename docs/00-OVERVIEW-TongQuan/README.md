# Tổng quan dự án QCVL

Điểm vào ngữ cảnh sản phẩm. Scope hiện hành ở [PROJECT-COORDINATION.md](../PROJECT-COORDINATION.md).

## QCVL là gì

QCVL là hệ thống vận hành nội bộ của Xưởng Quảng Cáo Văn Lâm: bán hàng, sản xuất, kho, tài chính và báo cáo.

## Người dùng

| Vai trò | Nhu cầu |
|---|---|
| Chủ xưởng | Vận hành, tài chính, báo cáo, cấu hình quan trọng. |
| Bán hàng/thu ngân | POS, khách, báo giá, hóa đơn, thanh toán. |
| Kho/vật tư | Tồn, nhập hàng, kiểm kho, cuộn/tấm. |
| Thợ máy/sản xuất | Hàng đợi và đối soát sản xuất. |
